# Ingrid VIP — Welcome Page

## Deploy na Vercel

1. Instale a Vercel CLI: `npm i -g vercel`
2. Na pasta do projeto: `vercel --prod`
3. Ou arraste a pasta no dashboard: https://vercel.com/new

## Personalizar

- `index.html` — todo o código do site
- `model.jpg` — foto da modelo (substituir pela foto desejada)
- `cards.png` — imagem das bandeiras de cartão
